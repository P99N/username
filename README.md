
# Username Search Tool made in python.



### Prerequisites:
Before using the script, make sure to install the necessary dependencies. You can install them via `pip`:

```
pip install requests colorama
```

### Usage and Example:
```
$ python main.py p99n
[+] Checking for username: p99n
```
